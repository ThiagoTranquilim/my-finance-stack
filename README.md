<div align=center>
  Aplicação fullstack para controle financeiro, desenvolvida em Python, FastAPI e PostgreSQL
</div>


## Objetivo

Permitir que o usuário tenha um controle maior da sua receita, despesas, assinatura, cartão e metas financeiras.

## Status do Projeto
🚧 Projeto em desenvolvimento.

## Como usar

```
git clone https://github.com/ThiagoTranquilim/my-finance-stack.git

cd my-finance-stack

python -m venv .venv
.venv\Script\activate

uvicorn app.main:app --reload
```

A API estará disponível em `http://localhost:8000`, a documentação estará disponível em `http://localhost:8000/docs`
